下記で、修正したmisakiにローカルのjarが置き換わる。 

$ cd ~/dev/misaki/
$ lein install

misaki本家が更新されたときは、 

$ git co master
$ git fetch upsutream
$ git merge upstream/master
$ lein install

こんな感じで。

もし修正版のmisakiを使用したくなくなったら リポジトリを手動で削除すれば良い(正式なやりかたがあるかもしれんが)。 

$ rm -rf ~/.m2/repository/misaki/misaki/0.2.6.2-beta
$ lein deps
