スペックの調べかた

Posted on 土曜日 2月, 11 2012 at 09:00:00.
# コア数(実環境)
$ cat /proc/cpuinfo | grep 'physical id'|wc -l

# コア数(仮想環境)
$ cat /proc/cpuinfo | grep processor|wc -l

# メモリ
$ free

# ディスク
$ sudo /sbin/fdisk -l

# OS
$ cat /proc/version

# OSのバージョンは下記でもよい
$ cat /etc/redhat-release
CentOS release 5.2 (Final)
