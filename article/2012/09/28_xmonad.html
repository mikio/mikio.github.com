xmonadを試す

Posted on 金曜日 9月, 28 2012 at 10:02:00.

CUIでtmuxを操作してるときの一覧できる快適さをGUIに求めた結果、自宅の ubuntuにxmonadをいれてみました。

  • http://gihyo.jp/admin/serial/01/ubuntu-recipe/0124

タイル型のウィンドウマネージャです。常にウィンドウ全てが一覧されるやつ です(Windows8は使ったことないけどタイル表示だから似てるのかな？でもキー ボード操作は全く考えらられてなさそう)。

それにしてもタイル型は快適ですな。普段からemacsでキーボード操作メインの ユーザだったら、すごく気にいるかと思います。とくにウィンドウ切り替えが Mod+j/kでできるのがいい!(それに比べて WindowsのALT+tabは左手の手首が死 ぬ。。。)唯一問題になりそうなのは、ゲームとかかな？(あ、あとGimpとかマ ウス主体でマルチウィンドウ的なアプリも使用しずらいだろうな。)

今回、Xmonadというのをインストールしましたが、老舗のawesomeというのも人 気があるそうです。ただXmonadの方が後発である分いろいろブラッシュアップ されてるような印象でした。問題は設定ファイルがHaskellってことですかね。 できれば覚えたくないですよね。いやHaskellは勉強して動かしたことはありま すが、今はLisp系言語を覚え中なので他の言語はできるだけ気にせず脳の負担 を増やしたくないのです。そういう意味で気になるのは、設定フィアルが CommonLispであるstumpwm。そのうち試したいと思います。

しかしタイル型使ってると、UnityとかgnomuShellの動向がどうでもよくなりま すな。

インストールメモ

$ sudo apt-get install xmonad
$ sudo apt-get install gmrun # キーワードランチャー
$ sudo apt-get install xmobar # ステータスバー

最低限のコマンド

Mod-S-qログアウト
Mod-S-cウィンドウクローズ
Mod-S-RET端末起動
Mod-S-pキーワードランチャー起動

~/.xmobar を作成する。 

Config { font = "xft:Sans-9:bold"
       , bgColor = "black"
       , fgColor = "grey"
       , position = Top
       , lowerOnStart = False
       , commands = [ Run Network "eth0" ["-L","0","-H","32","--normal","green","--high","red"] 10
                    , Run Cpu ["-L","3","-H","50","--normal","green","--high","red"] 10
                    , Run Memory ["-t","Mem: <usedratio>%"] 10
                    , Run Swap [] 10
                    , Run Battery [] 600
                    , Run Com "lsb_release" ["-s","-d"] "" 36000
                    , Run Date "%Y %b %_d(%a) %H:%M:%S" "date" 10
                    ]
       , sepChar = "%"
       , alignSep = "}{"
       , template = "<fc=#ee9a00>%lsb_release%</fc> | %cpu% | %memory% %swap% | %eth0% | %battery% }{ %date%"
       }

~/.xmonad/xmonad.hs を作成する。 

import XMonad
import XMonad.Hooks.DynamicLog
import XMonad.Hooks.ManageDocks
import XMonad.Util.Run(spawnPipe)
import XMonad.Util.EZConfig(additionalKeys)
import System.IO

main = do
        xmproc <- spawnPipe "/usr/bin/xmobar /home/mikio/.xmobarrc"  -- .xmoabrrcのパスを記述しておく
        xmonad $ defaultConfig
             { manageHook = manageDocks <+> manageHook defaultConfig
             , layoutHook = avoidStruts  $  layoutHook defaultConfig
             , logHook = dynamicLogWithPP $ xmobarPP
             { ppOutput = hPutStrLn xmproc
             , ppTitle = xmobarColor "green" "" . shorten 50
              }
             , modMask = mod4Mask     -- WindowsキーをModキーとしてバインドする
          } `additionalKeys`
          [ ((mod4Mask .|. shiftMask, xK_z), spawn "xscreensaver-command -lock")
          , ((controlMask, xK_Print), spawn "sleep 0.2; scrot -s")
          , ((0, xK_Print), spawn "scrot")
          ]
